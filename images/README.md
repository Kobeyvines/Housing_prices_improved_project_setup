## Saving Images

### Using the ImageSaver Class

```python
from src.visualization import ImageSaver

saver = ImageSaver(images_dir="images")

# Save a matplotlib figure
saver.save_figure(fig, "my_plot.png")

# Plot feature importance
saver.plot_feature_importance(importance, feature_names, filename="feature_importance.png")

# Plot predictions vs actual
saver.plot_predictions_vs_actual(y_true, y_pred, filename="predictions.png")
```

### Using utils functions

```python
from src.utils import save_plot

# Save matplotlib figure
save_plot(fig, "my_plot.png", images_dir="images")
```

### In Jupyter Notebooks

```python
# At the top of your notebook
%matplotlib inline
import matplotlib.pyplot as plt
from src.visualization import ImageSaver

saver = ImageSaver(images_dir="../images")

# Create and save plots
fig, ax = plt.subplots()
ax.plot(data)
saver.save_figure(fig, "my_plot.png")
```

## Available Visualization Methods

The `ImageSaver` class provides these plotting methods:

- `save_figure()` - Save any matplotlib figure
- `plot_feature_importance()` - Feature importance bar chart
- `plot_distribution()` - Histogram and box plot
- `plot_correlation_heatmap()` - Correlation matrix heatmap
- `plot_predictions_vs_actual()` - Prediction accuracy scatter plot
- `plot_training_history()` - Training/validation loss curves
- `plot_confusion_matrix()` - Classification confusion matrix
- `plot_metrics_summary()` - Model metrics bar chart

## Image Format & Quality

- **Format**: PNG (default), JPG, PDF, or SVG supported
- **DPI**: 300 (print quality)
- **Bbox**: Tight (removes whitespace)
- **Background**: White

## Git Configuration

Images are typically **not** committed to Git due to size. Add large images to `.gitignore`:

```
images/*.png
images/*.jpg
```

But commit important documentation images (e.g., architecture diagrams).

## Notebooks Integration

When creating visualizations in notebooks:

```python
# Import at top
from src.visualization import ImageSaver

# Initialize once
saver = ImageSaver(images_dir="../images")

# Save after creating plots
fig, ax = plt.subplots()
# ... create plot ...
saver.save_figure(fig, "notebook_visualization.png")
```

All plots will automatically save to this directory with proper naming and quality settings.
